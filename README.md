# B-U Marketing Portal

## 📌 Project Description
This is a web-based marketing portal developed for Bugema University. The system allows prospective and continuing students to explore academic programs, apply for admission, and submit applications for scholarships, bursaries, and work programs.

It also collects feedback from applicants to evaluate marketing effectiveness.

---

## 🎯 Objectives
- Provide a centralized platform for program exploration
- Simulate a complete admission process
- Support different types of applications (scholarships, bursaries, work programs)
- Collect user feedback for analysis

---

## 👥 Target Users
- Prospective students
- Continuing students
- University administrators (simulated)

---

## 🚀 Features
- Program browsing (by school, department, level)
- Admission application (full + quick apply)
- Scholarship & bursary applications
- Work program applications
- Feedback system
- Responsive design

---

## 🔄 System Flow
1. User explores programs
2. Applies for admission
3. Submits application
4. System processes and returns status
5. User applies for scholarships/bursaries
6. Feedback is collected

## ⚠️ Note
This system simulates real-world university processes for academic purposes.
